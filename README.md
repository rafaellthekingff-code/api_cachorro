#  API de Cachorros

Aplicação web que consome uma API de imagens de cachorros utilizando PHP no back-end e JavaScript no front-end.

---

##  Funcionalidades

- Exibição de imagens aleatórias de cachorros  
- Busca por raça  
- Botão "Carregar mais"  
- Layout responsivo  
- Animação ao carregar os dados  

---

##  Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript  
- PHP  
- API pública  

---

##  API utilizada

https://dog.ceo/api/

A API retorna imagens aleatórias, por isso os resultados mudam a cada atualização.
